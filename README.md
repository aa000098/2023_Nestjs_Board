## Installation

```bash
# Nestjs cli 설치
$ npm i -g @nestjs/cli

# Nestjs 프로젝트 시작
$ nest new classum-sonhyunho

# nvm으로 node 버전 관리 
$ nvm install 18.19.1
$ nvm use 18.19.1

# TypeScript 설치
npm i -g typescript

# Nestjs 프로젝트 의존성 설치
$ npm install
```

## Running the app

```bash
# development
$ npm run start:dev

# production mode
$ npm run start:prod
```
